ÜK294 Frontend einer interaktiven Webapplikation realisieren

Im üK295 habe ich eine API zur Produktverwaltung eines Online-Shops umgesetzt. Damit können mittels HTTP-Anfragen Produkte und Kategorien erstellt, eingesehen, geändert, gelöscht und aufgelistet werden. Nun erhaltete ich im Rahmen üK294 den Auftrag, ein Web-Frontend zur bestehenden API zu entwickeln. Dieses soll eine komfortable Benutzerschnittstelle bieten, um die obengenannten Aktionen mit den Produkten und Kategorien ausführen zu können. Da die API mittels JWT geschützt wurde, muss das Frontend dem Benutzer eine Möglichkeit bieten, sich bei der API zu authentifizieren und ein Token zu erwerben. Ohne Authentifizierung kann der Benutzer nicht auf die Daten zugreifen. Die Applikation muss das MVC-Entwurfsmuster umsetzen.

Um Sie sich authentifizieren zu lassen, klicken Sie auf Login in die Navigation.
